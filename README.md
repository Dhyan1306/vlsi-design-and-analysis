# vlsi-design-and-analysis

This repository showcases a collection of digital circuits designed and simulated during my Digital VLSI laboratory coursework. The work focuses on understanding CMOS design techniques and analyzing the behavior of both combinational and sequential circuits through simulation and layout implementation.

The project includes the complete design flow, starting from schematic creation and simulation to performance analysis and physical layout design.

Circuits Designed

- CMOS Inverter
- Symmetric CMOS Inverter
- NAND Gate
- NOR Gate
- XNOR Gate
- Full Adder using Mirror Logic
- 4:1 Multiplexer using Pass Transistor Logic
- Tri-State Inverter
- MUX-Based D Flip-Flop
- D Flip-Flop using Transmission Gate
- D Flip-Flop using Tri-State Inverter
- Domino Logic
- Domino Logic with Keeper Circuit
- Pass Transistor Logic
- Transmission Gate Logic

Highlights

- CMOS-based digital circuit design
- Transient response analysis
- Propagation delay evaluation
- Rise and fall time measurements
- Noise margin analysis
- Sequential circuit implementation
- Dynamic logic design
- Layout generation and verification

Tools and Technology

- Cadence Virtuoso
- Spectre Simulator
- CMOS 180nm Technology

Project Goal

The objective of this project was to gain practical experience in VLSI design by implementing commonly used digital circuits and studying their performance characteristics. It provided hands-on exposure to circuit design, simulation, timing analysis, and layout development using industry-standard tools.
